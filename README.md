# IHR Hub Projects

Welcome to the IHR Hub Projects repository! This is home to multiple Illinois Human Resources web projects, including accessible documents, training modules, and future applications. This guide will help you get started contributing, whether you're adding a document, updating a training module, or working on a new app.

## What's in This Repository?

The IHR Hub Projects hosts multiple HR-related web projects:

- **Documents** - Accessible HTML versions of HR documents
- **Trainings** - Interactive course modules and training materials
- **Future Projects** - Space for new web applications and tools as they're developed

**Live Site:** https://ihr-hub.illinois.edu


## Repository Structure

```
ihr-projects/
├── documents/              # HR documents project
│   ├── _template.html     # Template for new documents
│   ├── policies/
│   ├── procedures/
│   └── handbooks/
│
├── trainings/             # Training modules project
│   ├── _template.html     # Template for training pages
│   ├── module-1/
│   ├── module-2/
│   └── assets/           # Shared training resources
│
├── [future-projects]/     # New projects go here
│
├── .nojekyll             # Tells GitHub not to use Jekyll
├── CNAME                 # Custom domain configuration
└── README.md            # You are here!
```

Each project folder is self-contained with its own templates, styles, and resources.
